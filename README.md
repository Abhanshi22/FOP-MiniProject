# FOP Mini Project

## Description
This repository contains the C programming mini project developed for the course Foundations of Programming (FOP) (Semester 2) at MIT World Peace University (MIT-WPU).
The project demonstrates the application of fundamental programming concepts through a complete working solution, along with supporting documentation and presentation materials.

#📘 Academic Details
Course: Foundations of Programming (FOP)
Semester: 2
Institution: MIT World Peace University (MIT-WPU)
Language Used: C Programming

#Project Overview
This mini project focuses on strengthening core programming skills such as:
Problem-solving and logic building
Conditional statements and decision making
Looping constructs
Functions and modular programming
Arrays and structured data handling
Basic file/project organization
