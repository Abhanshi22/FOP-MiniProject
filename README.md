FOP Mini Project

This repository contains the C programming mini project developed as part of the Foundations of Programming (FOP) course (Semester 2) at MIT World Peace University (MIT-WPU).
The project demonstrates the implementation of fundamental programming concepts through a complete working solution, along with supporting documentation such as PPT, poster, and report.

📘 Academic Details

Course: Foundations of Programming (FOP)
Semester: 2
Institution: MIT World Peace University (MIT-WPU)
Language Used: C Programming

🧠 Project Description

This mini project is designed to apply core programming concepts learned during the course into a practical implementation. It focuses on developing logical thinking and structured programming skills using C.

Key concepts implemented:

Problem-solving and logic building
Conditional statements (if-else, switch-case)
Looping constructs (for, while, do-while)
Functions and modular programming
Arrays and basic data handling
Structured program design
C Programming
